Setting up a FlickrMirror

using flickrmirror.

Let's set up a mirror

We'll need to do the following things:

  1. have a git client (github has one for your OS if you don't already)
  2. install nodejs if you haven't already, from nodejs.org
  3. clone flickrmirror from github
  4. have a Flickr API key and secret. If you don't have these, a non-commercial API key is free over at Flickr's API page.

If you have (done) all these things, we can get started setting up a flickr mirror.

make sure all the dependencies are installed

flickrmirror$> npm install

set up an .env file

flickrmirror$> copy env.dist .env

open up this .env file, and drop in your Flickr API key and secret

You can also fill in the screen name you want to use, but this is used only once the mirror has been set up, and isn't technically used for fetching data from Flickr.

first-time run

flickrmirror$> node app --downsync

This will start up the downsycn process, but the app, with your API key, so really YOUR app, now needs to be authorised to access all your data.

...

real run

flickrmirror$> node app --downsync

Now sit back: all your stuff is download to your own drive for you do do with as you please.

serving up your mirror

flickrmirror$> node app

Open a browser and point it to localhost:3000 and presto. Your own mirror of your own Flickr data.
